## 💻 Especificaciones de Software

Para el desarrollo y validación de la etapa actual, se utilizaron las siguientes herramientas:

* *Simulador:* [Wokwi](https://wokwi.com) - Se utilizó para realizar el esquema de conexiones virtual y probar la lógica de los servos antes del montaje físico.
* *Lenguaje de Programación:* C++ (entorno Arduino).
* *IDE:* Arduino IDE 2.0.
* *Librerías:* Servo.h para el control de señales PWM de los motores MG995.
