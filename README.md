# Garra Robótica Controlada por Joysticks

Descripción breve: Este proyecto consiste en el diseño, ensamblaje y programación de una garra robótica de 4 grados de libertad, operada en tiempo real mediante joysticks analógicos.

---

##  Enlaces Rápidos a Entregables
* 📁 [Código Fuente (src)](./src/) - Scripts de control y optimizaciones.
* 📁 [Documentación (docs)](./docs/) - Diagramas de bloques y manual de montaje.
* 📁 [Hardware](./hardware/) - Esquemas electrónicos y lista de materiales (BOM).
* 📁 [Pruebas (tests)](./tests/) - Casos de validación del sistema.

---

##  Estructura del Repositorio
Siguiendo el estándar del proyecto, el repositorio se organiza de la siguiente manera:

* **`src/`**: Contiene el código fuente para el microcontrolador (Arduino/ESP32).
* **`docs/`**: Diagramas de flujo, esquemas de conexión y manual de usuario.
* **`hardware/`**: Archivos de diseño de piezas (STL para 3D) y esquemas electrónicos.
* **`results/`**: Mediciones de precisión y tiempos de respuesta de los servomotores.

---

##  Requisitos Previos
Antes de empezar, asegúrate de tener:
1. Microcontrolador (ej: Arduino Uno / Nano).
2. [X] Servomotores (ej: MG996R o SG90).
3. 2 Joysticks analógicos.
4. Fuente de alimentación externa (recomendado 5V-6V).
